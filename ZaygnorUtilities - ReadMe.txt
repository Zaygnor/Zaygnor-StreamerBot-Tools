Importing this will create the utility actions listed below in StreamerBot.

IF IN CHAT MESSAGE
Search each chat message to see if the value set in the global variable is found in the message. If found triggers the "if in chat message - found" action.

IF IN CHAT MESSSAGE - FOUND
A simple method to do whatever you want if the match is found. Right now it just spits out a message to the chat but you could have it trigger a sound or graphic or whatever.
